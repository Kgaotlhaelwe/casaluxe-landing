Questions

Useful answers before purchase.

Does CasaLuxe include Shopify checkout?

CasaLuxe supports Shopify's native checkout, dynamic checkout buttons and accelerated checkout containers. Payment providers are configured in Shopify admin.

Can buyers customize the theme?

Yes. CasaLuxe uses Online Store 2.0 sections, theme blocks and global settings for colors, typography, buttons, cards, spacing, header, footer and cart options.

Are demo products included?

Yes. The package includes a demo product CSV and documentation. Buyers should replace demo content with their own products and licensed imagery before launching.

Does it require jQuery?

No. CasaLuxe uses vanilla JavaScript and avoids heavy frontend dependencies.

What is the live demo password?

Use rtawst if Shopify asks for the storefront password when opening the live demo.
