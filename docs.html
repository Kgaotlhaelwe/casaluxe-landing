Shopify Theme Documentation

CasaLuxe setup guide.

Install the theme, configure Shopify settings, customize storefront sections and prepare your furniture or home decor store for launch.

Installation

Upload the theme ZIP in Shopify admin.

  1. 01

    Open Shopify admin and go to Online Store > Themes.

  2. 02

    Choose Add theme, then Upload zip file.

  3. 03

    Upload the installable CasaLuxe theme ZIP.

  4. 04

    Open Customize to configure sections, colors, typography, menus and footer content.

Customization

Use Shopify Theme Editor for visual setup.

  • Set brand colors, typography and button style.
  • Configure header navigation and mega menu content.
  • Edit hero slides, shop-by-room cards, collection blocks and lookbook sections.
  • Assign product metafields for dimensions, materials, care, delivery and assembly.
  • Configure Shopify payments, shipping, taxes and pickup in Shopify admin.
  • Use Shopify Search & Discovery for collection filters.
  • Replace demo imagery and product content before publishing.
  • Test cart, checkout, customer accounts and localization settings before launch.

Support

Before requesting help.

Please include your store URL, theme version, browser/device, screenshots, steps to reproduce and whether third-party apps are involved.

What support includes

Installation guidance, locating theme settings, demo content guidance, documented feature clarification and bug fixes for the delivered theme files.

What support does not include

Custom development, product data entry, third-party app setup, payment provider approval, custom design services or image licensing.
